# Fontes de dados (principais)

- UCDP (Uppsala Conflict Data Program) — battle-related deaths, conflitos.
  https://ucdp.uu.se/

- ACLED — eventos políticos e de violência (registro de protestos, repressões).
  https://acleddata.com/

- UNODC — homicide statistics.
  https://dataunodc.un.org/

- SIPRI — military expenditure.
  https://www.sipri.org/

- UNHCR — refugiados e deslocados.
  https://www.unhcr.org/

- World Bank — população (para normalizar per 100k), indicadores sociais.
  https://data.worldbank.org/

- Gallup — law & order / well-being surveys.
  https://www.gallup.com/

Observação: ACLED pode requerer registro e token de acesso para downloads via API.
